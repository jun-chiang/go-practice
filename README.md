# go-practice

## 字节跳动Go后端训练营日常练习



### 模块说明

#### guessGame
```
猜数字小游戏，随机生成一个 0~100的数字，让用户来猜
每次都提示用户猜大了或者猜小了，只到猜测正确
包含对各种非法情况的处理
```
#### wordTranslate
```
从控制台实现单词查询
从控制台输入一个单词，在控制台输出单词以及音标
在下方输出单词的翻译
支持循环输入单词查询，输入#退出程序
```
#### goroutine
```
使用协程来实现并发，协程的使用可以避免切换线程时巨大的系统开销
通过使用通道或者共享内存的方式来实现不同协程之间的通信
在使用共享内存的方式来通信时需要注意资源的协程安全问题
```
